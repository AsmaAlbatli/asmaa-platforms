# منصات المشرفة أسماء الباتلي

مجموعة أدوات تعليمية وإشرافية تفاعلية تعمل مباشرة في المتصفح بدون أي تثبيت.

## المحتوى

- **`index.html`** — الصفحة الرئيسية للوصول إلى الأدوات.
- **`platform.html`** — منصة تحليل الاختبارات ونتائج الطلاب (تحليل إحصائي، رسوم بيانية، تحليل فقرات، طباعة وتصدير Excel).
- **`quiz.html`** — منصة اختبار الرخصة المهنية لتخصص الإشراف التربوي (بنك أسئلة تدريبية مع تصحيح فوري ومراجعة).

## طريقة الاستخدام

افتح `index.html` في أي متصفح حديث، أو استخدم رابط النشر (GitHub Pages) ومشاركته مع الآخرين.
